Accept payments on your Wordpress site via credit card or Papara account. 

== Description ==

Papara payment plugin, enable merchants to accept payments easily from their customers' credit card or Papara account on Woocommerce-based sites. With this add-on, you'll be able to solve your web site's payment problem with Papara's easy and secure payment system as well as a constantly updating and evolving software.

= WHY CHOOSE PAPARA? =

There is no monthly fee or setup fee for Papara. You will never encounter surprises. You will be charged only when you earn money. Also you do not need to wait if you want to take your earnings. You can draw your money whenever you want.

Instantly distribute / send payments to thousands of people with the Papara Mass Payment API or by uploading files from your member merchant portal.

Papara is also for people who do not want use a bank or credit card, and therefore as a member merchant you can increase target customers.

= FEATURES OF PAPARA =

If you want to use Papara Payment plug-in, you should enroll as a merchant via Papara. Therefore you can take advantage of all the opportunities that a Papara account can provide.

* You can load cash to your account from 15 banks and more than 40,000 ATMs.

* You only need to know mobile phone or email address to send money to someone.

* There is no processing fee and no expense, 24/7 freedom in Papara!

== Installation ==

= Minimum Requirements =

WordPress 4.4 or greater

* Wordpress part of installation

= Automatic installation = 

If you don't want to bother yourself with downloading and uploading plug-in file you can easily install plug-in via Wordpress Dashboard. Log in to your dashboard and navigate to Plugins menu and click Add New.

Search for "Papara" and simply click "Install Now" 

= Manual installation = 

You can find further information about manual installation on [Wordpress Codex](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation "Wordpress Codex")

* Papara part of installation

To use Papara Payment Method on your site, you should be enrolled as a Merchant at [Member Merchant Portal](https://papara.com/#merchant "Become a Merchant")

After installing the plugin and becoming a merchant some configuration should be made.

1. Go to Woocommerce -> Settings -> Checkout section and find Papara.
2. There are some text field need to be filled. Some of them have default values.


```
Enable/Disable : In order to enable Papara Payment Module, check box.

Title: This controls the title which the user sees during checkout.

Description: This controls the description which the user sees during checkout.

Api Key: Given to Merchant by Papara, can be found in the merchant account, api information section.

Secret Key: In order to secure payments, can be found on Papara Merchant Account, also at api information section.

Test Mode: In order to make tests, check box. This way transactions can be made in order to sure about a working system. 

```
After getting values and entering correctly to the plugin setting page, it is highly recommended to test whether plugins works flawlessly. Don't forget to unmark test option when going live.

== Frequently Asked Questions ==

=  Does this plugin work with credit cards or just Papara? =

This plugin supports payments using both credit and debit cards as well as Papara.

=  Do I have to wait when I want to draw money from my Papara account as a member merchant? =

No, you can draw your money at anytime.

= How can a user buy things in my store using Papara account? =

Users can shop on your site by uploading money to their account in various ways. They can use ATM's or they can transfer money from their bank account. Or they can send money from another Papara account with phone number or email.

= Does this plug-in has a test mode? =

Yes, it can be activated via plugin settings.

= Will this plugin work with my theme? =

Yes, Papara Payment plugin works with all themes.
